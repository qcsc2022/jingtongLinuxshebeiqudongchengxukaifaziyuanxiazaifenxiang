### 内容简介
本书是Linux设备驱动程序开发领域的权威著作。全书基于2.6内核，不仅透彻讲解了基本概念和技术，更深入探讨了其他书没有涵盖或浅尝辄止的许多重要主题和关键难点，如PCMCIA、I2C和USB等外部总线以及视频、音频、无线连网和闪存等驱动程序的开发，并讲解了相关的内核源码文件，给出了完整的开发实例。

本书适合中高级Linux开发人员阅读。
